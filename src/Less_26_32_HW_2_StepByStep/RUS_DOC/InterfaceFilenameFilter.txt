******* Interface FilenameFilter *******

Экземпляры классов, реализующих этот интерфейс, используются для фильтрации
имен файлов. Эти экземпляры используются для фильтрации списков каталогов в
методе list класса File и компонентом файлового диалога Abstract Window Toolkit.

******* Метод *******

- boolean accept(File dir, String name) - Проверяет, должен ли указанный файл
                                          быть включен в список файлов.

Параметры: dir - каталог, в котором был найден файл;
           name - имя файла.
Возвращает: true тогда и только тогда, когда имя должно быть включено в список
            файлов, в противном случае false.
