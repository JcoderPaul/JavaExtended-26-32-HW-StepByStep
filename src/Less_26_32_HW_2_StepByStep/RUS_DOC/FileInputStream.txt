******* Class FileInputStream *******

public class FileInputStream extends InputStream

Все реализованные интерфейсы:
Closeable, AutoCloseable

FileInputStream получает входные байты из файла в файловой системе.
Доступные файлы зависят от среды хоста. FileInputStream предназначен
для чтения потоков необработанных байтов, таких как данные изображения.
Для чтения потоков символов лучше использования FileReader.

******* КОНСТРУКТОРЫ *******
FileInputStream(File file) - Создает FileInputStream, открывая подключение
                             к реальному файлу, файлу, названному файлом
                             объекта File в файловой системе.

FileInputStream(FileDescriptor fdObj) - Создает FileInputStream, используя
                                        файловый дескриптор fdObj, который
                                        представляет существующее подключение
                                        к реальному файлу в файловой системе.

FileInputStream(String name) - Создает FileInputStream, открывая подключение к
                               реальному файлу, файлу, названному по имени пути
                               в файловой системе.

******* МЕТОДЫ *******
int available() - Возвращает оценку количества оставшихся байтов,
                  которые можно прочитать (или пропустить) из этого
                  входного потока без блокировки при следующем вызове
                  метода для этого входного потока.

void close() - Закрывает данный входной поток файла и освобождает все
               системные ресурсы, связанные с потоком.

protected void finalize() - Гарантирует, что метод закрытия данного входного
                            потока файла вызывается, когда на него больше
                            нет ссылок.

FileChannel getChannel() - Возвращает уникальный объект FileChannel, связанный
                           с текущим (данным) входным потоком файла.

FileDescriptor getFD() - Возвращает объект FileDescriptor, представляющий подключение
                         к фактическому файлу в файловой системе, используемой этим
                         FileInputStream.

int read() - Считывает байт данных из этого входного потока.

int read(byte[] b) - Считывает до b.length байтов данных из данного (текущего) входного
                     потока в массив байтов.

int read(byte[] b, int off, int len) - Считывает до len байтов данных из этого входного
                                       потока в массив байтов.

long skip(long n) - Пропускает и отбрасывает n байтов данных из входного потока.
