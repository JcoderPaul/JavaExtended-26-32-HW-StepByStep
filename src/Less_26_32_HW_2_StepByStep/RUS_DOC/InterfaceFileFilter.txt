******* Interface FileFilter *******

Фильтр для абстрактных путей. Экземпляры этого интерфейса
могут быть переданы в метод listFiles(FileFilter) класса File.

******* Метод *******

- boolean accept(File pathname) - Проверяет, следует ли включать
                                  указанный абстрактный путь в
                                  список путей.

Параметры: pathname - абстрактный путь для проверки;
Возвращает: true тогда и только тогда, когда имя пути должно быть включено.
